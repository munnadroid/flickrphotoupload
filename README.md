flickrphotoupload
=================

Flickr Photo Upload in Android Phonegap using zepto and javascript.

Tested
--------
- Android 4.0.3
- Cordova 2.1.0


Uses
--------
- cordova 2.1.0
- zepto.min.js
- javascript

How to make it works ?
----------------------
- Enter appkey in #30 of script.js
- Enter secretkey in #31 of script.js
- Enter authtoken in #32 of script.js
